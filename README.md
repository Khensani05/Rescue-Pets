🐾 Pet Adoption Web Application 📌 Overview

This is a full-stack web application that allows users to:

View pets available for adoption Post pets for adoption Adopt pets Make donations toward a fundraising goal

The system simulates a real-world pet adoption platform with dynamic filtering and database-driven functionality.

🛠️ Technologies Used C# ASP.NET MVC SQL Server (Database-First Approach) LINQ & Lambda Expressions jQuery (DOM manipulation) Bootstrap

⚙️ Features 🐶 View and filter pets by type, breed, and location 📄 View detailed pet information ❤️ Adopt pets through a form ➕ Post new pets for adoption 💰 Donate and track progress toward a goal 🖼️ Upload and display pet images

🗄️ Database Setup (IMPORTANT FOR YOU) ‼️The database is included in the repository and must be restored or attached before running the application.‼️

Download the RescuePets.sql file
Open it in MS SQL
Change the connection string in the WebConfig file to your local database server
Open the solution file in visual studio
Clean and Rebuild
Run the code
